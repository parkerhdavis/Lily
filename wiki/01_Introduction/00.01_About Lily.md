# Lily Documentation

**Lily is a legal document drafting and client management tool.** It streamlines the process of gathering client information, populating Word document templates, and producing polished first drafts ready for attorney review.

Think of it as a bridge between client intake and document drafting — you fill in the details once, and Lily handles the rest.

## How Lily Works

Legal professionals often start client engagements by producing a set of standard documents — powers of attorney, living wills, engagement letters, and so on. These documents share a common structure but need to be customized with each client's specific information.

**Lily's approach:**
1. You organize work by **client folder** — one folder per client
2. You pick **templates** — standard Word documents with placeholders like `{Client Full Name}`
3. Lily copies the template into the client folder and lets you **fill in the blanks** through a form
4. A **live preview** updates as you type, so you can see exactly what the finished document will look like
5. When you're done, the completed `.docx` is ready to open in Word for final review and polishing

Information is shared across documents for the same client, so you only enter a client's name, address, and phone number once — every document that uses those fields picks them up automatically.
