# Installation

This guide covers installing Lily and getting it running for the first time.


## Download Lily

Download the latest release from [GitHub Releases](https://github.com/parkerhdavis/Lily/releases).

---


## Windows Installation

1. Download the Windows installer (`.msi` or `.exe`) from the releases page
2. Run the installer and follow the prompts
3. Launch Lily from the Start Menu or desktop shortcut

**Note:** Windows may show a SmartScreen warning for unsigned applications. Click "More info" then "Run anyway" to proceed. Lily is safe to install.

---


## Linux Installation

Lily is available in three formats for Linux:

### AppImage (Recommended)

The AppImage is a universal binary that works on most Linux distributions.

1. Download the `.AppImage` file from the releases page
2. Make it executable:
   ```bash
   chmod +x Lily_*.AppImage
   ```
3. Run it:
   ```bash
   ./Lily_*.AppImage
   ```

### Debian/Ubuntu (.deb)

For Debian-based distributions (Ubuntu, Linux Mint, Pop!_OS, etc.):

```bash
sudo dpkg -i Lily_*.deb
sudo apt-get install -f   # install dependencies if needed
```

### Fedora/RHEL (.rpm)

For RPM-based distributions:

```bash
sudo dnf install ./Lily-*.rpm
```

---


## macOS Installation

1. Download the `.dmg` file from the releases page
2. Open the `.dmg` and drag Lily to your Applications folder
3. On first launch, right-click the app and select "Open" to bypass Gatekeeper

---


## System Requirements

**All platforms:**
- 2 GB RAM
- 100 MB free disk space

**Windows:** Windows 10 version 1809 or later

**Linux:** WebKitGTK 4.1+, GTK 3+

**macOS:** macOS 10.15 (Catalina) or later

---


## First Launch

When you first open Lily, you'll land on the **Lily Hub** — the app's home screen. Before you can start working with clients, you'll want to set your templates folder:

1. Navigate to **App Settings** (gear icon or via the grid menu)
2. Under **Templates Folder**, click **Browse** and select the folder where your Word document templates are stored
3. That's it — you're ready to go

If you don't have templates yet, see [Understanding Templates](wiki/04_Templates/03.00_Understanding%20Templates.md) for how to create them.

## Troubleshooting Installation

### Windows: SmartScreen warning

Click "More info" then "Run anyway." This is normal for unsigned applications.

### Linux: AppImage won't run

Make sure FUSE is installed:
```bash
# Ubuntu/Debian
sudo apt install fuse libfuse2

# Fedora
sudo dnf install fuse fuse-libs
```

### macOS: "App cannot be opened"

Right-click the app, select "Open," and confirm in the dialog. This only needs to be done once.
