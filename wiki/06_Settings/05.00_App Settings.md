# App Settings

Lily's settings are accessible from the grid menu or directly from the Lily Hub's Settings card.

---


## Appearance

Toggle between light and dark modes. You can also use the sun/moon button in any screen's header. Your preference persists across sessions.

| Theme | Description |
|---|---|
| **Light** | Light background, dark text |
| **Dark** | Dark background, light text |

---


## Zoom

Adjust the interface scale from 50% to 200%. The zoom level is shown in the status bar.

| Shortcut | Action |
|---|---|
| `Ctrl++` / `Cmd++` | Zoom in |
| `Ctrl+-` / `Cmd+-` | Zoom out |
| `Ctrl+0` / `Cmd+0` | Reset zoom to 100% |

---


## Status Bar Size

Choose the height of the status bar at the bottom of the app: **Small**, **Medium**, or **Large**.

---


## Autosave

When enabled (the default), the Variable Editor automatically saves your changes after you stop typing for 2 seconds. When disabled, you must save manually with `Ctrl+S`.

---


## Workspace

The Workspace section configures where Lily looks for your files.

### Client Libraries

One or more directories containing client folders. Lily scans these directories to populate the Clients Hub, which shows all clients across your configured libraries with document status tracking.

Click **Add** to add a library directory, or remove existing ones.

### Template Libraries

The folder where your Word document templates (`.docx` and `.dotx`) are stored. Lily scans this folder to populate the Template Picker and Pipeline views.

Click **Browse** to select your templates folder.

### Questionnaire Libraries

The folder where questionnaire definition files are stored. This defaults to the app's config directory, but can be changed to any location.

---


## Where Settings Are Stored

Lily stores its settings in your operating system's standard config directory:

- **Windows:** `%APPDATA%\lily\`
- **Linux:** `~/.config/lily/`
- **macOS:** `~/Library/Application Support/lily/`

This includes:
- `settings.json` — Your preferences (library paths, theme, zoom, window size, etc.)
- `logs/` — Application log files (daily rolling, useful for troubleshooting)

You generally don't need to interact with these files directly.

## Keyboard Shortcuts

Lily supports the following keyboard shortcuts:

### Document Editing
| Shortcut | Action |
|---|---|
| `Ctrl+S` / `Cmd+S` | Save document |
| `Ctrl+Z` | Undo |
| `Ctrl+Y` / `Ctrl+Shift+Z` | Redo |

### Navigation
| Shortcut | Action |
|---|---|
| `Alt+Left` | Navigate back |
| `Alt+Right` | Navigate forward |

### Search
| Shortcut | Action |
|---|---|
| `Ctrl+F` / `Cmd+F` | Focus the search field (in Variable Editor and Questionnaire) |

### View
| Shortcut | Action |
|---|---|
| `Ctrl++` / `Cmd++` | Zoom in |
| `Ctrl+-` / `Cmd+-` | Zoom out |
| `Ctrl+0` / `Cmd+0` | Reset zoom |
| Theme toggle button | Switch light/dark mode |

A keyboard shortcut reference is also accessible from the keyboard icon in the status bar at the bottom of the app.
