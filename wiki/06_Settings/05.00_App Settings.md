# App Settings

Lily's settings are accessible from the grid menu (⊞) or directly from the Lily Hub.

---


## Templates Folder

The most important setting: the folder where your Word document templates are stored. Lily scans this folder to populate the Template Picker and Pipeline views.

Click **Browse** to select your templates folder. You can change this at any time — the Template Picker will immediately reflect the new location.

---


## Theme

Lily supports light and dark modes. Toggle between them using the sun/moon button available in every screen's header, or change it in App Settings.

| Theme | Description |
|---|---|
| **Light** | Light background, dark text |
| **Dark** | Dark background, light text |

Your theme preference is saved and persists across sessions.

---


## About

The About section shows Lily's current version number and other application information.

---


## Where Settings Are Stored

Lily stores its settings in your operating system's standard config directory:

- **Windows:** `%APPDATA%\lily\`
- **Linux:** `~/.config/lily/`
- **macOS:** `~/Library/Application Support/lily/`

This includes:
- `settings.json` — Your preferences (templates folder, theme, last working directory)
- `questionnaires/` — Questionnaire definition files
- `logs/` — Application log files (daily rolling, useful for troubleshooting)

You generally don't need to interact with these files directly.

## Keyboard Shortcuts

Lily supports the following keyboard shortcuts:

### Document Editing
| Shortcut | Action |
|---|---|
| `Ctrl+S` / `Cmd+S` | Save document |
| `Ctrl+Z` | Undo |
| `Ctrl+Y` / `Ctrl+Shift+Z` | Redo |

### Navigation
| Shortcut | Action |
|---|---|
| `Alt+Left` | Navigate back |
| `Alt+Right` | Navigate forward |

### View
| Shortcut | Action |
|---|---|
| Theme toggle button | Switch light/dark mode |

A keyboard shortcut reference is also accessible from the keyboard icon in the status bar at the bottom of the app.
