# The Pipeline

The Pipeline is Lily's management area for templates and workflow configuration. While the Client Hub focuses on a single client's documents, the Pipeline gives you a bird's-eye view of your template library and client setup tools.

---


## Accessing the Pipeline

Open the Pipeline from the grid menu (⊞) in any screen's header. You can switch between the Pipeline and a client without losing your client context.

---


## Template Browser

The Pipeline's main view shows your templates folder as a browsable tree. You can expand folders, click templates to inspect them, and see which variables each template contains.

This is useful for:

- **Auditing templates** — See what variables a template uses before assigning it to a client
- **Checking consistency** — Verify that templates use the same variable names for the same information
- **Planning** — Get an overview of your template library

---


## Client Setup

The Pipeline includes a **Client Setup** section for managing the tools used during client intake:

### Questionnaire Definitions

The primary tool in Client Setup is the questionnaire definition manager. From here you can create, edit, duplicate, and delete questionnaire definitions — the structures that define what questions appear in the client questionnaire.

See [Managing Questionnaires](wiki/05_Pipeline/04.01_Managing%20Questionnaires.md) for details.
