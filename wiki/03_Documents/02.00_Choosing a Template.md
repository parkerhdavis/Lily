# Choosing a Template

When you need to add a new document to a client folder, the Template Picker lets you browse your templates and select one to use.

---


## Opening the Template Picker

From the **Client Hub**, click **Add New Document** at the bottom of the document list. The Template Picker opens, showing your configured templates folder as a navigable folder tree.

---


## Browsing Templates

Templates are displayed in the same folder structure they have on disk. Click a folder to expand it, and click a template to select it.

### Search and Filter

If you have a lot of templates, use the search field at the top to filter by file name. The tree collapses to show only matching templates and their parent folders.

### "In Use" Indicators

Templates that already have a document in the current client folder are highlighted in green with an "in use" badge. This helps you see at a glance which templates have already been applied for this client.

---


## Selecting a Template

Click a template to select it. Lily copies the template `.docx` into the client folder and opens it in the Variable Editor, where you can begin filling in variables.

### If the Template Is Already in Use

If you select a template that already has a document in the client folder, Lily shows a dialog with two options:

- **Open existing document** — Opens the document that's already in the client folder, with its saved variable values loaded
- **Create a new copy** — Makes a fresh copy of the template, creating an additional document

This prevents accidentally creating duplicate documents while still allowing it when you need multiple copies of the same template.

---


## Where Templates Come From

Templates are standard Word documents (`.docx`) with placeholder variables written in curly braces — like `{Client Full Name}` or `{Date of Birth}`. They live in a folder you configure in [App Settings](wiki/06_Settings/05.00_App%20Settings.md).

For more on how templates work and how to create them, see [Understanding Templates](wiki/04_Templates/03.00_Understanding%20Templates.md).
